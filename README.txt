miraFONT - COMPLETE KIT
=======================
Everything for the miraNET Digital Runtime typeface.

/fonts       21 cuts x 3 formats (.otf .ttf .woff) in Dots/ Linear/ Spline/
             + miraFONT.css (ready @font-face, every cut wired)
/esp32       ANSI C font + SSD1306 OLED driver + Arduino & ESP-IDF demos (ELEGOO 0.96")
miraFONT-specimen.html   full interactive specimen + live OLED simulator (works offline)
MAKE-WOFF2.txt           one command to also generate the optional .woff2 web format

INSTALL FONTS
  Windows  double-click a .ttf or .otf -> Install
  macOS    double-click -> Install Font (Font Book)
  Linux    copy the .ttf/.otf to ~/.local/share/fonts/  then:  fc-cache -f
  Web      use fonts/miraFONT.css

CUTS
  Weights : Light, Regular, Bold, Black
  Styles  : Italic, Compressed, Narrow, Double Strike
  Textures: Dots (node matrix), Linear (merged stroke), Spline (smooth monoline)

Built from a 5x7 node matrix, no third-party outlines.
